# QR Code Generator

A simple and responsive QR Code Generator built using **HTML, CSS, and JavaScript**. It allows users to generate QR codes instantly for any text or URL using the QR Server API.

## 🚀 Features

- Generate QR codes instantly
- Supports both text and URLs
- Clean and responsive UI
- Input validation with shake animation
- Uses QR Server API (No backend required)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QR Server API


## 📂 Project Structure

```
QR-Code-Generator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ⚡ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/QR-Code-Generator.git
```

2. Open the project folder.

3. Run `index.html` using Live Server or any browser.

4. Enter any text or URL and click **Generate QR Code**.
